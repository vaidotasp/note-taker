# Note-Taker Application

### Description

A simple note taking application built in order to learn API routing, MongoDB interactions and Pug template engine.

Project Link:
(https://note-taker-app.glitch.me/)

Backend:
* NodeJS/Express
* MongoDB/Mongoose
* Mocha/Chai

Frontend:
* Pug for templating
* VanillaJS for everything else
